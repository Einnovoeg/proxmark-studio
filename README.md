# Proxmark Studio

## Status
Proxmark Studio is an actively maintained project for working with the Proxmark RFID/NFC reader.

## Features
- Comprehensive command set for Proxmark operations
- Easy-to-use GUI for visual feedback and control
- Support for multiple platforms

## Build/Run Instructions
1. Ensure you have the required dependencies installed.
2. Clone the repository:
   ```
   git clone https://github.com/Einnovoeg/proxmark-studio.git
   cd proxmark-studio
   ```
3. Build the project:
   ```
   make
   ```
4. Run the application:
   ```
   ./proxmark-studio
   ```
