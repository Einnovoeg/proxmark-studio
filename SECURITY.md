# Security Policy

## Reporting a Vulnerability
If you believe you have found a security vulnerability in this project, please report it to us as follows:

1. **Email us**: Send an email to security@einnovoeg.com.
2. **Details to include**:
   - A description of the vulnerability
   - Steps to reproduce it
   - Any potential impact

We will take your report seriously and look into it as soon as possible.

## Security Policy
We take security seriously and aim to respond to vulnerability reports within 7 days. We will acknowledge your report and keep you updated during our investigation.

Thank you for helping keep this project secure!
